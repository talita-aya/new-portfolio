DONUT

Depois que finalizar meu primeiro projeto no Figma com animação, era preciso tentar novas coisas diferentes... Foi assim que, assistindo um tutorial no Youtube este projeto pode se tornar real!

A parte de posicionar os elementos na tela, torná-los visualmente bonitos e escolher quais imagens seriam utilizadas foi a parte mais simples do projeto (na minha opinião), porque segue o mesmo modelo de planejamento que é realizado para desenvolver outras ideias. Deste modo, a paleta de cores de baseou, exclusivamente, nos donuts escolhidos.

A escolha tipográfica também foi simples, visto que haveria pouco texto escrito e a escolha não serifada se encaixaria melhor com uma animação alimentícia.

BEBAS NEUE, fonte única

O grande problema surgiu na hora de adicionar ações para criar a sensação de movimento suave e bem feito. Entretanto, mesmo seguindo o tutorial utilizado, algo não funcionava corretamente na minha prototipação, algo que mais tarde descobrir ser elementos visuais estáticos como plano de fundo e que foram retirados para que não prejudicassem o projeto completo.

Este é o típico projeto simples, mas com o qual aprendi novas funcionalidades e novas técnicas para prototipar elementos, de modo a gerar um maior encantamento no usuário e melhorar sua experiência.

Para ver a animação acontecer basta clicar no donut e arrastar para baixo ou para cima!
