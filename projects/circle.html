CIRCLE

Este projeto tem uma abordagem totalmente diferente do que eu fazia até então, pois senti a necessidade de aprimorar minhas habilidades e criar telas para sistemas web com o objetivo de praticar a utilização de diversas ferramentas que o Figma possui e aprender coisas novas.

Desenvolvido com base em um tutorial do Youtube, o Circle é um dashboard para controle e visualização de informações de uma empresa de vendas, contando detalhes que muitas vezes passam despercebidos, como o detalhe que mostra se o usuário está online e a presença de notificações não lidas.

A escolha de cores seguiu a discrição mas trouxe elementos vivos para prender a atenção em determinadas informações, como é o caso do violeta e do rosa.

A escolha tipográfica manteve-se no básico e padrão: Poppins, visto que para este tipo de trabalho, ter uma fonte facilmente legível é imprescindível para facilitar a leitura e evitar desentendimentos.
